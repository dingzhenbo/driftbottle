# flutter 社交app 集成环信IM

一个集成环信IM的Flutter项目。目前只是Android与flutter双向交互， 已完成功能：注册、登录、退出登录、自动登录、切换头像、切换主题、搜索好友、关注好友、 取消关注、发送文本消息 还有很多模块待开发。。。
